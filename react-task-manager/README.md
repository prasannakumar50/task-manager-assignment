# ⚛️ React Task Manager

A simple **Task Manager application** built using React and Bootstrap.

Users can:
-  Add tasks
-  Mark tasks as completed
-  Delete tasks
-  Filter tasks (All / Completed / Pending)

---

#  Project Setup (Step-by-Step Guide)



##  Step 1 — Clone the repository

git clone <your-repo-url>

## Step 2 — Navigate to React project
cd react-task-manager

## Step 3 — Install dependencies
npm install

## Step 4 — Start development server
npm run dev

## Features Implemented

- Add new task
- Delete task
- Toggle completion
- Filter tasks
- Component-based architecture

## Tech Stack

- React
- JavaScript / JSX
- Bootstrap
- Jest + React Testing Library

## Components

UI is split into small reusable components:
- TaskForm → add tasks
- TaskList → list tasks
- TaskItem → single task
- FilterBar → filtering
- State Management
- State is managed using:
- React useState
- Props drilling between components
